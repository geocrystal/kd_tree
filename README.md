# Kd::Tree

Crystal implementation of "K-Dimensional Tree" and "N-Nearest Neighbors"
based on http://en.wikipedia.org/wiki/Kd-tree.


## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  kdtree:
    github: mamantoha/kd_tree
```

## Usage

```crystal
require "kd_tree"
```
